redaction_gone_wrong

PDF Redaction Copy/Paste

Goal: Find the “redacted” key that wasn’t actually removed from the PDF.

Challenge Description

Screenshot from the picoCTF challenge page:

Challenge Overview

We’re given a PDF report where some text is blacked out. The catch: the “redactions” were done incorrectly. Instead of removing the underlying text, the author basically just covered it up — meaning the hidden text can still be selected and copied.

Solution

1) Open the PDF and test the redactions

I opened the report and tried selecting the text around the black bars. Even though it looks hidden visually, the text underneath is still present. Copying and pasting that “redacted” area reveals the flag.
