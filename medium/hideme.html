Overview

We’re given a PNG (flag.png) that looks like a normal picoCTF logo. The description hints that there’s “more than what meets the eye,” which usually means there’s extra data appended or embedded in the file.

The plan: use binwalk to scan for hidden content, extract it, then navigate to find the flag.

Solution

1) Detect + extract embedded data with binwalk

First, I scanned the PNG with binwalk to identify hidden content. The output showed embedded data containing a secret/ directory and a hidden image. In the same step, I ran binwalk -e to extract the embedded files. 

binwalk flag.png
binwalk -e flag.png

2) Navigate to the extracted directory

After extraction, binwalk created an output folder (commonly named something like _flag.png.extracted). Inside it, the secret/ directory contained the hidden flag.png. 

ls
cd _flag.png.extracted/
ls
cd secret/
ls

3) Open the recovered image

Opening the extracted secret/flag.png reveals the flag.

Flag

picoCTF{Hiddinng_An_imag3_within_@n_ima9e_d55982e8}

Tools Used

  • binwalk (detect embedded data)
  • binwalk -e (extract embedded files)
  • Basic shell navigation: ls, cd
