Goal: Find the hidden picoCTF{...} flag inside a very large text file using the known prefix.

Challenge Description

Screenshot from the picoCTF challenge page:

Challenge Overview

The download is a huge text file (anthem.flag.txt) with a ton of filler content. The hint says to search using the known prefix — so instead of scrolling forever, we just grep for picoCTF{.

Solution

1) Search the file for the flag prefix

Since we know flags start with picoCTF{, we can grep for that string and pull the line instantly: 

cat anthem.flag.txt | grep picoCTF

Flag

picoCTF{gr3p_15_@w3s0m3_58f5c024}

Tools Used

  • cat (view file contents / pipe into grep)
  • grep (search by known prefix)
