SVG Ctrl+A Copy/Paste Hidden Text

Goal: Find the tiny hidden flag text inside the SVG and reconstruct it (remove the spaces).

Challenge Description

Screenshot from the picoCTF challenge page:

Challenge Overview

The file we’re given is drawing.flag.svg. Since it’s an SVG (vector image), the “image” can contain real text objects — including text that’s so small you can’t see it normally.

The easiest way to reveal tiny text in an SVG is to select everything (Ctrl + A), which highlights all objects on the page, including the hidden flag.

Solution

1) Open the SVG and select all

Open drawing.flag.svg in your browser. Then press Ctrl + A. This highlights the tiny text near the center. Copy it (Ctrl + C) and paste it into a text editor.

2) Remove the spaces

The copied flag appears with spaces between characters. Remove the spaces to get the real flag.
