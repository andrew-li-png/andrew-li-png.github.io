Goal: Extract the hidden flag from the image using LSB steganography analysis.

Challenge Description

Screenshot from the picoCTF challenge page:

Challenge Overview

The description doesn’t give much, but the challenge name and category point straight to steganography. For PNGs, zsteg is a quick way to scan common LSB/MSB bit-planes and immediately spot embedded text.

Solution

1) Run zsteg on the image

After downloading the provided PNG, I ran zsteg to check for hidden data in the bit planes. The flag shows up directly in the output. 

zsteg pico.flag.png
