Solution

This challenge is a straightforward introduction to SleuthKit. We are given a disk image and asked to find the size of the Linux partition using mmls.

After downloading and extracting the disk image, I ran: 

mmls disk.img

The output clearly shows a Linux partition with a length of 202752 sectors.

After launching the instance on the picoCTF page, a checker service is provided. I connected using netcat and entered the partition size: 

nc saturn.picoctf.net 63770

Submitting the correct value returns the flag immediately.

Flag: picoCTF{mm15_f7w!}

Tools Used

  • mmls (SleuthKit)
  • nc (netcat)
